# Bank Churn Analysis

## 📌 Project Overview
This project analyzes customer churn in a bank using data analysis techniques. The goal is to identify key factors that lead to customer attrition and provide actionable insights to improve customer retention.

---

## 🎯 Objectives
- Identify customers who are likely to churn
- Analyze factors affecting churn rate
- Provide data-driven recommendations to reduce churn

---

## 🛠️ Tools & Technologies Used
- SQL (Data Extraction & Transformation)
- Power BI (Dashboard & Visualization)
- Excel (Data Cleaning)

---

## 📊 Key Insights
- Customers with low account balance are more likely to churn
- Customers with fewer products tend to leave more frequently
- Certain age groups show higher churn rates
- Geography plays a role in customer retention

---

## 📈 Dashboard Features
- Churn Rate Overview
- Churn by Age Group
- Churn by Balance
- High-Risk Customer Identification
- Interactive Filters (Gender, Geography, Age Group)

---

## 🚀 Project Workflow
1. Data Cleaning using SQL/Excel
2. Data Transformation
3. Data Modeling in Power BI
4. Creating Interactive Dashboard
5. Generating Insights

---

## 💡 Business Recommendations
- Target high-risk customers with personalized offers
- Improve engagement for customers with fewer products
- Focus retention strategies on high churn regions
